## 为什么 No-Contact 是基础，但通常远远不够

No-Contact-Regel 大概是分手后被重复得最多的建议。它很简单：一段时间内不主动联系。不写消息、不打电话、不在社交媒体上互动。

这一招有三层作用：你给 自己 一个从情绪冲击里缓过来的机会；你让对方真正感受到你的缺席；你避免因为不断打扰而把分手决定在对方心里越刻越深。

但 No-Contact 只是地基，不是整栋房子。如果你在沉默几周后突然又以老样子出现，什么都不会改变。真正起作用的是沉默期间你做了什么准备，以及沉默之后你如何重新登场。

心理触发器（Trigger）就是在这里登场的。Parship 的指南强调：不要过早表达复合愿望，而是要等对方自己产生好奇。Focus 的"十步指南"也指出关键——走出受害者角色，不要追着对方跑。很多人恰恰死在这一步：要么太早联系，要么方式错了。

Heiko Rieger 在博客文章《Die Psychologie des Vermissens》中解释，想念并非偶然，而是遵循可识别的模式。理解这些模式并加以利用，比单纯寄希望于沉默自动解决问题要靠谱得多。

## Fernhypnose 与 Remote Viewing 到底是什么

这两套方法是 Hypnoticum Mentalisticus 的核心，但它们究竟是什么？两者都属于争议性技术，缺乏广泛的科学支持。

**Fernhypnose** 指的是不依赖物理在场、从远距离尝试影响一个人的潜意识。课程使用催眠语言模式、Suggestionstechniken 和 Konditionierung 方法，意图在潜意识层面化解对方的情感障碍。

**Remote Viewing** 最早是一项军事技术——冷战期间为美国情报机构开发，用于"看见"和描述远距离目标。Heiko Rieger 通过他的公司 Psychic Entertainment 把这项技术民用化，用于分析情感关系模式。按他的说法，他是"民用领域应用军事级 Fernwahrnehmung 的先锋"。

**Remote Influencing** 再往前一步：不只是感知，还要有意识地在远距离施加影响。这是整套方法里概念上最具争议的部分，因为它暗示了对另一个人施加主动作用，这在伦理上会引发疑问。

老实说，这些方法在科学上争议很大，目前没有可靠的独立研究能证实其有效性。Heiko Rieger 提到的 92%–95% 成功率和超过 7,000 对伴侣，都来自他自己的陈述，无法独立核实。相信心理影响可能性的人，可能在这里找到价值；想要科学证据的人，更适合转向经典伴侣咨询。

那为什么这门课依然有它的拥趸？因为分手的绝望里，很多人愿意尝试一切看起来还有希望的东西——而这门课恰恰提供了那种"联系已经断了、对方根本不配合"情境下还能用上的工具。传统辅导在这些场景里很快就到底了。
